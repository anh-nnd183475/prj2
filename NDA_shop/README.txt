Sau khi tải hoặc clone source code về:
hãy mở file config.php để config mySQL 
Tải xampp để chạy localhost.
sau đó mở admin trong xampp.
trong mySQL, chọn nhập, tải file lên; sau đó tải file :
"db_shopbanhang" lên, hãy để ý trong file config có tên như thế nào thì setup tên cơ sở dữ liệu như thế.
sau đó vào browser, nhập đường dẫn:
"localhost/NDA_shop/"- để tới trang index
và "localhost/NDA_shop/" để tới trang admin
tên đăng nhập và password của admin nằm trong cơ sở dữ liệu, hãy xem trước khi đăng nhập admin.
